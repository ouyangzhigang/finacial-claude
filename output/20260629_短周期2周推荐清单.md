- **指数位置**:上证 4073.90(+1.16%)、深成 15812.87(+0.19%)、创业板 4216.70(+0.54%)、**科创50 2126.01(+4.61% 领涨)**。

### 2. 货币与流动性环境(🟢充裕)
- **央行**:降准降息周期延续(宽货币),3.5 万亿成交 = 增量资金入场迹象明显。
- **资金面**:两市 3.5 万亿、换手榜大面积 25-40% 高换手(半导体材料端集体高活性)、涨停潮 → 游资+机构共振。
- **北向资金**:周末休市取不到实时净流入(接口返回空),但成交额能级 + 科创50 大涨指向外资/机构流入科技。**数据缺失,以成交额代理**。

### 3. 国际形势与地缘(🟡中性,外部冲击源已在催化日历)
- **美联储**:6 月已暂停,7/2 会议纪要 + 7/3 非农落窗 → 美元/美债波动传导北向与成长股估值,但 A 股科技主线由国产替代逻辑主导,外部扰动有限。
- **中美科技管制**:半导体/EDA/光刻管制持续 → 反而强化**自主可控/国产替代**主线(大基金三期投向设备/材料/设计),利半导体内循环。
- **传导路径**:美元↑→北向阶段性流出→成长股波动;但当前内资主导(3.5 万亿),科技主线韧性强。

### 4. 政策与产业风向(🟢主线明确)
- **半导体**:大基金三期(投向设备/材料/HBM/先进封装)+ 国产替代政策 → 设备(中微 +10.58%)、材料(康强/中巨芯/飞凯集体高换手)、设计(澜起 +13%)全产业链主升。
- **创新药**:审批加速 + 医保谈判预期 + ASCO 管线催化 → 医药 ETF m5 +6.9% 刚启动(位置低于半导体)。
- **机器人/AI**:世界人工智能大会(7 月上海)+ 特斯拉 Optimus + 产业政策 → 机器人(300024)涨停。

### 5. 市场风格与情绪周期(🟢发酵)
- **风格**:科技成长(科创50/创业板/半导体)碾压大盘价值,小盘成长 + 题材占优。
- **情绪**:20cm 涨停潮集中在科创板/创业板半导体+创新药;换手榜 25-40% 高换手 = 人气发酵中(尚未到全面派发)。
- **主线锁定**(政策周期 + 市场热度双确认):
  - **方向 A:半导体全产业链(设备/材料/设计/封测)**——主升浪(ETF m5 +13.5%、m10 +38%、m20 +32%)
  - **方向 B:创新药/医药**——刚启动(ETF m5 +6.9%、m20 +3.5%,位置低更安全)
  - **方向 C:机器人/AI**——题材共振(涨停潮)

> **小结**:2 周窗口顺风方向 = **半导体(主矛)+ 创新药(盾型弹性)+ 机器人(题材进攻)**;价值/红利逆风,弱化底仓价值角色,改用**低价高流动性科技中军(京东方A)**代替底仓。
